# 국내 자동차 시장 트렌드 분석 – 대시보드 구성 및 기능 설계

이 문서는 Streamlit 기반 대시보드의 전체 구조 및 각 페이지 기능을 정의합니다. 주요 데이터 소스를 기반으로 한 관심도(수요)와 실제 판매량(보급률)의 차이를 시각적으로 보여주는 것을 목표로 합니다.

---

## 1. 전체 시장 트렌드 개요

**목적**  
국내 자동차 시장의 전체적인 흐름을 요약하여 사용자가 한눈에 파악할 수 있도록 제공합니다. 현대자동차와 기아자동차의 모델들을 중심으로, 관심도(검색량)와 보급률(등록 대수) 간의 차이를 시각화합니다.

**주요 기능**
- 전체 모델을 대상으로 한 월별 등록 대수 그래프
- 구글 트렌드, 네이버 검색량 데이터를 활용한 검색 관심도 추이
- 인기 모델의 보급률 및 관심도 간 상관관계 시각화
- 블로그 키워드 기반 워드 클라우드

---

## 2. 모델별 상세 분석

**목적**  
특정 모델을 클릭했을 때, 해당 차량에 대한 상세 정보를 시각적으로 제공합니다. 관심도와 보급률을 함께 보여주고, 사용자의 검색/구매 의사 결정 흐름을 파악하는 데 도움을 줍니다.

**주요 기능**
- 해당 모델의 월별 검색량 및 등록 대수 트렌드
- 연비, 가격, 연료 종류, 색상 등의 속성 정보 정리
- 등록 대수 대비 색상/연료별 분포
- 네이버 블로그 상위 3개 글 미리보기 및 핵심 키워드 워드 클라우드

---

## 3. 제조사별 모델 비교

**목적**  
현대자동차와 기아자동차 간의 경쟁 구도 및 모델군 비교를 통해 사용자에게 제조사 단위의 시각적 인사이트를 제공합니다.

**주요 기능**
- 제조사 선택에 따른 모델 필터링
- 각 제조사별 월별 등록 실적 추이
- 관심도 상위 모델 및 보급률 비교
- 가격대별 분포와 연료 유형 비율 등 통계 정보

---

## 4. 검색 트렌드 분석

**목적**  
모델명, 세그먼트, 연료 유형 등 다양한 키워드를 기준으로 검색 트렌드를 분석합니다. 구글 트렌드와 네이버 데이터랩을 함께 활용하여 관심도 흐름을 파악할 수 있도록 합니다.

**주요 기능**
- 키워드 입력 시 검색량 추이 시각화
- 경쟁 모델 간 상대 비교 기능
- 월별 관심도 변화 차트
- 검색량 급등 모델 자동 하이라이팅

---

## 향후 확장 가능성

- 세부 옵션(트림 제외)별 수요 파악
- 차량 색상, 연료, 가격대별 군집화 분석
- 커뮤니티 리뷰 기반 감성 분석
- AI 기반 검색량 예측 모델 적용
